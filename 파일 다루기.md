inode의 사용법을 완벽히 익혀야
파일을 제대로 다룰 수 있음.

파일 정보 탐색 -> [[stat, fstat(2)]]
파일 접근 권한 확인 -> [[access(2)]]
파일 접근 권한 변경 -> [[chmod, fchmod (2)]]

[[하드 링크]] / [[심벌릭 링크]]

하드 링크 생성 -> [[link(2)]]
심벌릭 링크 생성 -> [[symlink(2)]]
심벌릭 링크 정보 검색 -> [[lstat(2)]]
심벌릭 링크 내용 읽기 -> [[readlink(2)]]
심벌릭 링크의 원본 파일의 경로 탐색 -> [[realpath(3)]]
링크 끊기 -> [[unlink(2)]]
