호스트명 : 관리하기 쉽게 IP주소를 자연어로 치환한 이름
인터넷 호스트명 = "호스트명" + "도메인명"
www = 호스트명
oooo.co.kr = 도메인명
DNS = 호스트명과 도메인명 관리 시스템


/etc/hosts 에 데이터베이스로 호스트명이나 IP주소가 저장되어있음
```sh
hosts: files dns
```
files 에서 찾아보고, 여기서 못찾으면 DNS 서비스 이용

DNS 8.8.8.8
-> 여기가서 호스트명과 도메인명 검색
